■インストール概要

・LiveCD
Live CD（ライブCD）あるいはLive DVD（ライブDVD）とは、ハードディスクドライブやSSDなどの内部ストレージにインストールすることなく、直接OSを起動させることができるCDやDVDである。
https://ja.wikipedia.org/wiki/Live_CD
・USBWriter
イメージファイルを、USB メモリに書き込む
http://www.gigafree.net/media/writing/USBWriter.html

■yumアップデート

・アップデートが必要なパッケージを確認する
# yum check-update
# yum -y update
TIPS:mirrorlist.centos.orgに接続できない場合、他のサイトへ接続。以下ftp.riken.jpへ接続する場合 
# vi /etc/yum.repos.d/CentOS-Base.repo

[base]
#mirrorlist=http://mirrorlist.centos.org/?release=$releasever&arch=$basearch&repo=os&infra=$infra
baseurl=http://ftp.riken.jp/Linux/centos/$releasever/os/$basearch/

以下同様に変更する

[updates]
.
.
.
TIPS:必要な場合は以下も設定する 
# yum -y groupinstall "Base" "Development tools" "Japanese Support"
・yum-cronをインストールします。
# yum -y install yum-cron
・インストール済みのパッケージを毎日自動更新
# /etc/rc.d/init.d/yum-cron start
# chkconfig yum-cron on
# chkconfig --list yum-cron
・ランレベル345でon
yum-cron	0:off   1:off   2:on    3:on    4:on    5:on    6:off
参考：CentOS6 インストール後の設定 http://centos.server-manual.com/centos6_setup.html
