■mogoDB概要

■各種パス

・confファイル
/etc/mongod.conf
・実行ファイル
/etc/rc.d/init.d/vsftpd

■基本コマンド

・インストール
# yum insatll mongodb-org
・起動
# service mongod start
・自動起動
# chkconfig mongod on
・起動確認
# chkconfig mongod --list

■データベース操作

・ログイン
$mongo

・DB作成
$ mongo [dbname]
> use [dbname]
useコマンドで自動生成

・DB一覧[MySQL : SHOW DATABASE]
> show dbs

・DB削除 [MySQL : DROP DATABASE]
> use [dbname]
>db.dropDatabase()

■コレクション操作

・テーブル一覧 [MySQL : ALTER TABLE]
> show collections

・テーブル作成 [MySQL : CREATE TABLE]
> db.createCollection("[table]")

・テーブル削除 [MySQL : DROP TABLE]
> db.[table].drop()

■ドキュメント操作

・SELECT [MySQL : SELECT * FROM]
> db.[table].find()
